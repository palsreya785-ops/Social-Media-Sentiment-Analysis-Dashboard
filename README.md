# 📊 Social Media Sentiment Analysis Dashboard

## 🚀 Overview
This project is an end-to-end Machine Learning application that analyzes social media text (tweets/comments) and classifies sentiment into:

- 😊 Positive  
- 😐 Neutral  
- 😡 Negative  

It also provides an interactive dashboard to visualize sentiment distribution in real time.

---

## 🎯 Problem Statement
Businesses receive thousands of comments daily. Manually analyzing sentiment is:

- ❌ Time-consuming  
- ❌ Inefficient  
- ❌ Not scalable  

This project automates sentiment detection using NLP and Machine Learning.

---

## 💼 Industry Relevance
Used by companies to:
- Monitor customer feedback  
- Track brand reputation  
- Analyze product reviews  
- Improve customer experience  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- NLTK (Text preprocessing)  
- Scikit-learn  
- TF-IDF Vectorization
